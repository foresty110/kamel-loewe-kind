개요

철학을 좋아하는 사람들이 자유롭게 철학 주제를 나누는 커뮤니티의 자유게시판 목록 조회 화면이다.

화면 설명

1 카테고리
  • 카테고리는 DB에 저장되어 있음 (관리자 화면 없음)
2 검색
  • 제목 or 작성자 or 내용 텍스트 검색
  • 등록일 디폴트 최근 1년
3 건수
  • 검색된 게시물 총 건수
4 첨부 표시
  • 첨부파일이 있는 경우 아이콘 표시
5 제목
  • 80자 넘는 경우 '…'로 줄여서 표시
6 조회수
  • '보기'에서 증가시켜 줌
7 등록/수정 일시
  • 최초 등록일시 표기
  • 최종 수정일시 표기
  • 수정한 이력이 없다면 '-'로 표기
8 페이지
  • 페이지 이동, 한 페이지에 10개씩 노출
  • 페이지 이동 시 검색조건 유지 (보기·쓰기·수정에서도 마찬가지)
